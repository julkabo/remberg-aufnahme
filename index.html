So richtest du den Webhook ein

📋 Schritt-für-Schritt

1. script.google.com öffnen
2. „Neues Projekt" → Code aus der Apps-Script-Datei einfügen
3. „Bereitstellen" → „Neue Bereitstellung" → Typ: Web-App
4. Zugriff: Jeder → bereitstellen
5. URL kopieren und oben einfügen

⚠️ Bei „Zugriff von jedem" ist die URL öffentlich — behandle sie wie ein Passwort.
